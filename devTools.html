The Chrome DevTools Tips

Performance and Memory Leaks

We have 3 great tools for finding memory leaks in Chrome:

  • Chrome Task Manager
  • Performance Panel
  • Memory Panel

Elements

  • by press H , we can hide and unhide elements
  • by press Delete , we can Delete elements
  • Current selected element refers to the $0 on Console Panel. and modify like '$0.innerHTML="hi";' and $1 refers to an element before the current one and so on
  • on Computed Panel, we find the winning CSS style and why to win
  • DOM Breakpoints for finding problem of unwanted changing

Tips

  • By click on 3 dots beside Console Panel, click on Rendering and Paint flashing and ...
  • By hold Shift key on Network Panel,the green and red color appear
  • For keep our changes after Refresh page, we go to Sources Panel on Overrides
  • After use Performance, we can go to Sources and see the Runtimes of each items
  • On Sources, we add Conditional Breakpoint with Holding Ctrl key + Click
  • On Sources,we have IDE, if we add folders to Filesystem tabs
